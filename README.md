# Rilk: a Java 2D RPG

A simple 2D adventure RPG built entirely with vanilla Java libraries.  
Features several explorable areas, combat, story, NPC interactions, secrets and more.

## Features
- 2D tile rendering using Java Swing  
- Player movement and collision  
- Simple update/render loop
- Self-made art, music and sound effects
- A casino where you can gamble your (in-game) money

## Playing the Game
The game files can be downloaded for free [here](awsy.itch.io/rilk).
When trying to download the game you will be prompted if you wish to donate. 
This is completely optional, and can be skipped by clicking on "No thanks, just take me to the downloads".
